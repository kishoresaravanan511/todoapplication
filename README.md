# Todo Application

A full-stack **Todo Management Application** built using **React (Frontend)** and **Spring Boot (Backend)**.
This application allows users to create, view, complete, and delete tasks.

---

## 🚀 Features

* Add a new Todo
* View all Todos
* Mark Todo as completed
* Delete Todo
* Clean and responsive UI

---

## 🛠️ Tech Stack

### Frontend

* React
* Axios
* Material UI
* JavaScript
* HTML & CSS

### Backend

* Spring Boot
* Java
* REST API
* Maven

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```
git clone https://github.com/kishoresaravanan511/todoapplication.git
```

---

### 2️⃣ Run Backend

```
cd backend
mvn spring-boot:run
```

Backend will run on:

```
http://localhost:8080
```

---

### 3️⃣ Run Frontend

```
cd frontend
npm install
npm start
```

Frontend will run on:

```
http://localhost:3000
```
